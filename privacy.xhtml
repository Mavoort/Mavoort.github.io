Datenschutz

Datenschutzerklärung gem. DSGVO

Dies ist eine private Homepage zu nicht-kommerziellen Zwecken. Es werden von mir keine personenbezogene Daten erhoben oder gespeichert. Gehostet wird die Seite auf Github, einem Tochterunternehmen von Microsoft.
Aus technischen Gründen werden bei jedem Verbindungsaufbau folgende Daten übertragen:

  • IP-Adresse
    Diese ist im TCP/IP-Protokoll notwendig zur Übertragung der Webinhalte. Eine Webseite ohne IP-Adresse zu übetragen ist in etwa so, wie einen Brief zu verschicken, ohne die Ziel-Adresse draufzuschreiben.
  • User Agent
    Dieser beeinhaltet Informationen über Ihr Browser und Betriebssystem. Ihr User Agent lautet:
    JavaScript ist blockiert!

Amazon speichert diese Informationen in einer Logdatei.

Inhalte von Drittanbietern

Selbstverständlich wird die Verarbeitung von Daten auf das notwendige Maß beschränkt (“Datenminimierung”). Jedoch werden trotzdem einige Inhalte (Bilder, WebFonts, JavaScript-Bibioltheken, …) von Drittanbietern bereitgestellt, darunter:

  • KaTeX und MathJax: zur Darstellung mathematischer Formeln,
  • Google WebFonts: für schönere Schriften,
  • Wikimedia Commons: Bilder und Grafiken

Dadurch wird weniger Speicher verbraucht und somit Kosten gespart.

Cookies

Diese Seite verwendet Cookies. Diese sind zur Gestaltung der Homepage technisch notwendig und beinhalten keinerlei personenbezogene Daten. Ohne diesen Cookie würde Ihre Auswahl bzgl. des Stils der Seite (hell oder dunkel) beim Neuladen jedesmal zurückgesetzt werden. Ihr Cookie lautet:
Javascript ist blockiert!
Der Cookie hat eine Gültigkeit von 24 Studen und wird danach von Ihrem Browser automatisch gelöscht.

Besucherzähler

Auf der Startseite befindet sich ein Besucherzähler von flagcounter.com. Dieser gibt zwecks Reichweitenmessung Auskunft über die ungefähre Anzahl und Herkunft der Besucher. Bei dem Besucherzähler handelt es sich um nichts weiter als ein eingebundenes Bild. Es werden also nicht mehr Daten übertragen als beim Einbinden von anderen Bildern auch; lediglich IP und UserAgent, wie oben bereits erklärt. Dieses Bild wird nur beim Aufruf der Startseite geladen, es handelt sich also nicht um eine websiteübergreifende Nachverfolgung des individuellen Nutzerverhaltens (“Tracking”).

Videos

Des Weiteren werden auf dieser Seite YouTube-Videos eingebunden. Beim Laden der Videos wird von YouTube ein Cookie gesetzt. Falls Sie damit nicht einverstanden sind, so können Sie meine Homepage auch nutzen, ohne Videos anzuschauen. Welche Daten YouTube über Sie sammelt weiß ich auch nicht. Das können Sie aber in der Datenschutzerklärung von YouTube nachlesen.
